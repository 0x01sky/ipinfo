# IP Info Tool
an ip tool that displays full report or a partial report about your ip using free ipinfo.io api, purely written in bash!

# Tool Usage

```bash
Usage: ipinfo [OPTION]... [IP]
Description: Gets a partial or a full report about an IP or your IP.

            -c : Displays city and country from the IP.

            -r : Displays region from the IP.

            -o : Displays organization from the IP.

            -p : Displays the postal code from the IP.

            -t : Displays the timezone from the IP.

            -a : Displays a full report about the IP.
```

# Installation:

 - Method 1:

```bash
git clone https://github.com/0x01sky/ipinfo
cd ipinfo/bin && mv ipinfo $HOME/.local/bin/

```
- Methdo 2:

```bash
wget https://raw.githubusercontent.com/0x01sky/ipinfo/main/bin/ipinfo
mv ipinfo $HOME/.local/bin/

```
